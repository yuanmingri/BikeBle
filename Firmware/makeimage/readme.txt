################################################################################
###Use multi_ImageGenerate.bat for generating spi flash image with BD address

	>multi_genimg_bdaddr.bat arg1 arg2

		arg1 : image filename
		arg2 : BD Address
	e.g.
	>multi_genimg_bdaddr.bat fw_multi_part_spi_bd01 01:00:00:CA:EA:82
